# GameStore (Frontend)

A modern React Single Page Application (SPA) for an e-commerce game store, featuring a modular component architecture and enterprise-grade secure authentication.

## Security & Authentication

Read the full technical breakdown of how I secured this application with WSO2 Asgardeo on Medium.

> [How to Create a Secure Login in a React SPA with Asgardeo](https://sanjeewadev.medium.com/how-to-create-a-secure-login-in-a-react-spa-with-asgardeo-real-world-e-commerce-integration-962b49bb2460)

## Tech Stack

- **Framework:** React.js
- **Authentication:** Asgardeo React SDK

## Key Features

- Secure Login, Logout, and Session Management
- User self-registration flows
- Protected routing for authenticated users
